Phraseological elements included

kbd
Defines keyboard input
samp
Defines computer program output
code
Defines a piece of computer code
var
It is a tag that can be used to define or create variables or mathematical expressions on a web page.

Example:

kbd: Please press ctrl+s to save the file

sapm: I encountered this error this program does not open while running the Chrome program

code: This(meta charset="utf-8") is a piece of code from html

var:  The area of ​​the rectangle is h*x, where h is the length and x is the width of the rectangle
